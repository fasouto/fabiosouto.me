# fabiosouto.me

Personal site built with Astro.

## Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS 4](https://tailwindcss.com/) - Styling
- [MDX](https://mdxjs.com/) - Markdown with components
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Hosted on Cloudflare Pages. Pushes to `main` trigger automatic deployments.
